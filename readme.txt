== Description ==

This is very useful plugin that let your users know about post analytics means users can see how many time this post is read/watch.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. After activation the plugin you guys can find its setting page under word press settings tab with name 'J Post Views Counter'.
